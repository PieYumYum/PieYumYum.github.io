Lets first dive deeper into CSS

If we think of CSS in terms of selectors, properties and values, then a typical CSS entry would look like this

            
                selector {property: value;}

To style the above, we need to add 2 tags together. We call this CSS combinators

What are sudo classes?

Again, if we think of CSS in terms of selectors and properties and values, then a sudo class is an add-on to a selector

            
                selector:pseudo-class {property:value;}

We use them for links

If we want our link to look different then we can simply add it in my css as follows

            
                a {color: red;}

But if we want each of our link states to look different then we will use a sudo class like:

  • :visited
  • :hover
  • :active
